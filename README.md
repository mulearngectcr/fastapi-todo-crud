# Contributing Guidelines

Follow the steps below to submit your solution for this task.

---

## 1. Fork the Repository

Click the **Fork** button at the top-right of the repository, or use the link below:

```text
https://github.com/mulearngectcr/fastapi-todo-crud
```

This will create a copy of the repository under your GitHub account.

---

## 2. Clone Your Fork

Replace `<your-username>` with your GitHub username.

```bash
git clone https://github.com/<your-username>/fastapi-todo-crud.git
cd fastapi-todo-crud
```

---

## 3. Create a Folder with Your Name

Inside the repository, create a folder using your name.

Example:

```text
fastapi-todo-crud/
├── john-doe/
│   ├── main.py
│   ├── requirements.txt
│   ├── README.md
│   └── screenshots/
```

Place your complete solution inside **your folder only**.

You may also include:

- Screenshots
- Additional files
- Notes about your implementation
- Solutions to the bonus challenges

---

## 4. Commit Your Changes

```bash
git add .
git commit -m "Add submission - John Doe"
```

Replace `John Doe` with your name.

---

## 5. Push Your Changes

```bash
git push origin main
```

---

## 6. Create a Pull Request

1. Open your fork on GitHub.
2. Click **Contribute → Open Pull Request**.
3. Submit the pull request to the original repository.

---

## Pull Request Title Format

Use the following format:

```text
Submission - Your Name
```

Example:

```text
Submission - John Doe
```

---

## Important Notes

- Create **only one folder** using your name.
- Do **not** modify or delete another student's folder.
- Keep all of your files inside your own folder.
- Ensure your API runs successfully before creating the Pull Request.
- Feel free to attempt the bonus challenges.

Happy Coding! 🚀
